# Exploratory Testing of the Faktoria 3D Website https://druk3dkrakow.pl/

## **Test Charter**

**Identifier:** ET-F3D-01

**Objective:** Perform a comprehensive quality assessment of the website through exploratory testing. 

**Time:** I completed four testing sessions of 90 minutes each, focusing on validating the website’s functionality and behavior.

**Result:** I identified 16 defects that significantly impact the website’s professional appearance (including typos, content issues, and visual defects). All identified issues were reported using Jira.
